API Basejump: File Metadata Microservice

User stories:

  1. I can submit a FormData object that includes a file upload
  2. When I submit something, I will receive the file size in bytes within the JSON response.

Example usage:

https://secret-brushlands-95755.herokuapp.com/about

Upload file to https://secret-brushlands-95755.herokuapp.com/upload via post method.

Example output:

{ "size": "12345" }

File Input:

Select file to view size.
